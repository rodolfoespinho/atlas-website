# ATLAS — Engenharia Criativa

**Engineering the Future. Creatively.**

Website institucional da ATLAS — estúdio tecnológico especializado em engenharia de software, soluções SaaS e inteligência artificial.

## Stack

- HTML5 + CSS3 + Vanilla JavaScript
- Self-contained single-page application
- Zero dependências de build
- Google Fonts: Space Grotesk, Instrument Serif, DM Mono

## Deploy

Hosted on [Vercel](https://vercel.com). Auto-deploy on push to `main`.
